WEBGIS CÁC KHU BẢO TỒN THIÊN NHIÊN VIỆT NAM

Hệ thống bản đồ chuyên đề đang trong giai đoạn hoàn thiện dữ liệu và chức năng.

Tổng quan hệ thống

Kiến trúc hiện tại: Neon PostGIS làm kho dữ liệu, Backend API để CRUD/Logs, GeoServer để trả WMS/WFS, Leaflet để hiển thị.

Luồng dữ liệu

PostGIS (Neon) -> Backend API / GeoServer -> WebGIS.

Cách dùng nhanh

1) Mở tab Bản đồ để xem lớp dữ liệu.

2) Mở tab CRUD để thêm/sửa/xóa khu bảo tồn.

3) Mở tab Logs để theo dõi lịch sử chỉnh sửa.
